## Algoritmos Implementados

### 1. Priorización de Feed
```
R = (W_tipo * U) / ((D + 1)^2 * (T + 1))

Donde:
- W_tipo: Peso del tipo (Necesidades > Acciones > Actualizaciones > Directorio)
- U: Urgencia (0-10)
- D: Distancia en km
- T: Tiempo en horas
```

### 2. Privacidad de Ubicación
```
L_visible = {
  L_exacta si C(u1, u2) = Aceptada
  L_aprox(r) si C(u1, u2) ≠ Aceptada
}
```

### 3. Expansión Dinámica
```
if (usuarios < 500) {
  radio = 3km  // Expandir
} else if (usuarios > 5000) {
  radio = 0.5km  // Preparar para dividir
} else {
  radio = 1km  // Normal
}
```

### 4. Distancia Haversine
```
d = 2R * arcsin(√(sin²(Δlat/2) + cos(lat1) * cos(lat2) * sin²(Δlon/2)))
```
